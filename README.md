# Hadrian

A monorepo of security tools for finding vulnerabilities in web applications.

## Tools

| Tool | Description |
|------|-------------|
| [hadrian-api-tester](./hadrian-api-tester) | API security testing framework that tests for OWASP API Top 10 vulnerabilities using role-based authorization testing. Supports custom YAML templates, multiple output formats, and LLM-powered triage. |

See each tool's README for installation and usage instructions.
